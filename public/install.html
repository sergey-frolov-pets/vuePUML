Установка vuePlantUML (PWA)

Для установки с телефона нужны все файлы из архива и запуск через локальный HTTP-сервер. Открытие одного index.html через «Файлы» не регистрирует PWA.

Termux (на этом телефоне)

  1. Распакуйте архив, например в ~/PumlEditor
  2. В Termux выполните:
cd ~/PumlEditor
chmod +x start-termux.sh
./start-termux.sh
  1. Откройте в Chrome:
  2. Нажмите «Установить приложение» ниже или в меню Chrome (⋮)
  3. После установки запускайте с иконки на главном экране

Структура папки

index.html
manifest.webmanifest
sw.js
install.html
start-termux.sh
icons/
  icon-192.png
  icon-512.png
  icon.svg
  apple-touch-icon.png

Установка

Открыть редактор
Если кнопка «Установить» не появилась — используйте меню Chrome: ⋮ → «Установить приложение» / «Добавить на главный экран».
